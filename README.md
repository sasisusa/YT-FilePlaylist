# YT-FilePlaylist

Something to play a own created playlist from a file.

## Needed
* node.js
* Web browser

	if there is an error or problem with the opn-module, or should they not be found: 
	`$ npm install opn` or global shared `$ npm install -g opn`
  
## Run
`$ node yt-fpl.js`
